# Implement Next Slice

Implement the next **[TODO]** slice from the development plan.

## Steps

1. **Understand the plan**: Read `plan/README.md` for project context and goals
2. **Find the next task**: Read `plan/TASKS.md`, locate the first `[TODO]` task
3. **Read slice details**: Open `plan/03-incremental-implementation-slices.md`, find the matching slice
4. **Review specs**: Check `plan/01-ux-and-behavior.md` for UX requirements relevant to this slice
5. **Review implementation notes**: Check `plan/02-implementation-notes.md` for technical guidance
6. **Implement**: Create/modify files listed in **Touchpoints**—stay focused, no unrelated changes
7. **Verify**: Follow the **Manual Checkpoint** steps exactly; fix issues until it passes
8. **Update status**: Change task from `[TODO]` to `[DONE]` in `plan/TASKS.md`

## Rules

- One slice only—don't combine or skip ahead
- Checkpoint must pass before marking done
- If blocked, stop and explain the issue
